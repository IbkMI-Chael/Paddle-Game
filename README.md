# Paddle Game

A classic two-player Pong game built with Python's Turtle graphics library.

## Technologies Used
- Python 3
- Turtle (built-in Python library)

## How to Run It
1. Make sure you have Python 3 installed
2. Download ALL these files into the same folder:
   - main.py
   - paddle.py
   - ball.py
   - scoreboard.py
3. Open your terminal in that folder
4. Run: python main.py

## How to Play
- **Right Player:** Arrow Up / Arrow Down keys to move
- **Left Player:** W (up) / S (down) keys to move
- First player to score wins the point
- Ball speeds up as the game progresses

## Features
- Two-player gameplay on the same keyboard
- Ball collision detection with walls and paddles
- Automatic scoreboard tracking
- Ball resets after each point
